# 2.3 索引和查詢
```
PUT /customer/external/1?pretty
{
  "name": "John Doe"
}
```

輸出
```
{
  "_index": "customer",
  "_type": "external",
  "_id": "1",
  "_version": 1,
  "result": "created",
  "_shards": {
    "total": 2,
    "successful": 1,
    "failed": 0
  },
  "created": true
}
```

查詢：
```
GET /customer/external/1?pretty
```
輸出
```
{
  "_index": "customer",
  "_type": "external",
  "_id": "1",
  "_version": 1,
  "found": true,
  "_source": {
    "name": "John Doe"
  }
}
```
