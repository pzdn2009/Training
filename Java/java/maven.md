# Maven

MAVEN是基于POM Project Of Model的构建管理工具。

- 安装
- 存储库
- 依赖
- unit test

使用aliyun镜像：

```xml
    <mirror>  
        <id>nexus-aliyun</id>  
        <mirrorOf>*</mirrorOf>  
        <name>Nexus aliyun</name>  
        <url>http://maven.aliyun.com/nexus/content/groups/public</url>  
    </mirror>
```
